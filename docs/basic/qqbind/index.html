🔖绑定QQ

# 1. 在游戏内申请绑定

进入服务器，输入以下命令

/bd QQ号
1

假如你的QQ号是 114514，那么你需要发送的命令是

/bd 114514
1

# 2. 在QQ群中确认绑定

在QQ群中发送以下消息

绑定 玩家名
1

假如你的名字是 LittleCatX，那么你需要发送的是

绑定 LittleCatX
1

可以艾特机器人也可以不艾特，艾特错了机器人不会触发确认，所以建议不艾特。
如果没反应再艾特机器人

# 3. 领取奖励

请按照QQ机器人提示，回到服务器中，在主城找到冒险者协会，在里面找到NPC夏娜，即可将头衔升级至铜锭！
