🗺️跨服地图画

普通的地图物品无法在主服区、生电区、迷失领域之间穿梭，否则地图的内容会根据该服区的地图数据而改变。

如果需要让地图物品跨服，请将地图放入铁砧，在结果处拿出带有跨服地图画描述的物品即可，该操作将会把地图内容存到物品中，并锁定地图。
